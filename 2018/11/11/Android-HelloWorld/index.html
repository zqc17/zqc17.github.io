从零开始--Android HelloWorld

发表于 2018-11-11 | 分类于 Android

本文简单介绍如何创建第一个Android工程Hello World。

安装及配置Java环境

参考Java菜鸟教程，完成Java, JDK的安装以及JAVA_HOME, CLASSPATH, PATH等环境变量的配置。

安装Android Studio

注意这一步需要科学上网，否则需要一些额外处理。

下载并安装Android Studio，按默认配置一直下一步就好了。当网络状况良好的情况下第一次启动的时候会自动下载Android SDK等依赖。

参考Android Studio的安装与配置创建第一个Android工程Hello Worlld，过程中可能会遇到以下问题：

  • gradle连接超时。通过 File->Settings->Build,Exception,Deployment->Build Tools->Gradle->Android Studio ， 选中 Enable embbedded Maven repository 。
  • 导出的apk默认没有签名无法在手机上正常安装。 <工程目录>/app/build/outputs/apk/debug 下的apk文件可以使用（release下的apk文件默认没有签名）。
  • 新建虚拟设备（用于调试）的时候网络不好会加载不出来system image下载链接。
  • 新建虚拟设备后还需要到 File->Settings->System Settings->Android SDK->SDK Tools 安装 HAXM installer(Intel x86 Emulator Accelerator)才可以正常开启虚拟设备。
  • 配置工程加载apk到虚拟设备以方便调试。
