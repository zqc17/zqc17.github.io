从零开始--基本Android UI

发表于 2018-11-25 | 分类于 Android

本文简单介绍Android的基本UI组件，参考菜鸟教程。

常用组件

Android API提供的组件中开发中常用的组件如下：

  • TextView。文本框，用于显示文本的组件。
  • EditView。输入框，显示一个用于输入文本的输入框。
  • Button, ImageButton。按钮，图像按钮，显示一个按钮。
  • ImageView。 图像视图，用于显示图片的组件。
  • RadioButton。 单选按钮，显示一个单选按钮。
  • CheckBox。 复选框，显示一个复选框。

布局

Android API提供的布局有6种，如下，但是头两种最为常用：

  1. LinearLayout
  2. RelativeLayout
  3. TableLayout
  4. FrameLayout
  5. GridLayout
  6. AbsoluteLayout
